- Plasma theme
- Aurorae theme
- Color scheme

![Screenshot](preview.png)

# Install
```
git clone https://github.com/varlesh/Arc-Dark-KDE.git
cp -R Arc-Dark-KDE/{aurorae,color-schemes,plasma} ~/.local/share/
```
